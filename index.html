FIND-DUPLICATES

find duplicate files into specific directory


Introduction

FIND-DUPLICATES - this is a simple console tool which will find duplicate files into specified directory. Additionally it have a small set of options which can be provided via command line and allow you to customize it behavior.

Files are considered duplicates if:

  1. They are the same size;
  2. They have the same checksum;
  3. They have the same content.
By default, the program will recursively scan the specified directory and search for files of the same size and with the same checksum. This is the fastest way to find duplicates and can be used in many cases. But with a very small probability it may occurs that unequal files will be accepted as duplicates (If it will have the same size and checksum). If you are analyzing important data and you need to make sure that the files are really duplicated, you can specify an additional option and then the program will perform an additional step and compare the candidates in pairs. Pairwise comparisons can take significantly much time, so you should keep that in mind.

Application

The default use of the program is simple - just enter its name and the directory to be scanned:


  find-duplicates /path/to/directory

  ToDo: Here should be program output sample

Also you can specify more than one directory for scanning


  find-duplicates /path/to/directory1 /path/to/directory2 /path/to/directory3

  ToDo: Here should be program output sample

For a guaranteed comparison, add the option -c


  find-duplicates -c /path/to/directory

  ToDo: Here should be program output sample

Additionally find-duplicates supports the following options:

ShortLongDescription
-h--helpWill show help and exit
-v--verboseThe program will show extended messages
-c--compareDuplicate candidates will be compared pairwise
